# TheMarketKilla Web Site

Landing page moderna y profesional para **TheMarketKilla**, marca de trading automatizado de Criptomonedas y Forex.

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- lucide-react

## Correr en local

```bash
npm install
npm run dev
```

Luego abre [http://localhost:3000](http://localhost:3000).

## Deploy en Vercel

### Opcion 1: desde la web

1. Sube este repo a GitHub.
2. Entra a [Vercel](https://vercel.com/new).
3. Importa el repositorio.
4. Manten la configuracion por defecto de Next.js y despliega.

### Opcion 2: desde CLI

```bash
npm i -g vercel
vercel
```

Para produccion:

```bash
vercel --prod
```
